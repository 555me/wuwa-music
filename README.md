# 仓库信息/Repository Infomation
本仓库为鸣潮游戏音频（除角色语音外）更新记录仓库。<br/>
This repository is used for Wuthering Waves audio update history but not include character voices.<br/>
<br/>
您可以通过本仓库的JSON文档或Pak6文件中，WW-G/Event文件夹的BNK文件获取WEM文件的具体音频名称。<br/>
Your can search WEN file's name info by JSON file in this repo OR BNK files in WW-G/Event which be export from Pak6 at game files.<br/>
<br/>
推荐通过<b>github客户端</b>或<b>VS Code</b>查看完整历史（自2.7.9 正式服版本起）。<br/>
We recommend your that use <b>Github Desktop</b> OR <b>VS Code</b> to view full history from Ver. 2.7.9 (Live).<br/>
<br/>
请自行寻找处理本仓库中WEM文件的方法，本仓库暂不提供。<br/>
We'll don't supply the way about how to play the WEM files normally, so you must search it by yourself.
